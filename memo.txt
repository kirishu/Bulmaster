Visual Studio Codeでの使い方
    必要な拡張機能
        npm, SCSS Intellisence

    最初にやること
        Ctrl + @ : npm install

    毎回起動時にやること
        Shift + Ctrl + B : build-sass 実行

----------------------------------------------------------
menuタグは1つだけであること
mainタグは1つだけであること
section.containerも1つだけであること

main
    余白無し
    サイドメニューのresponseve設定
    navやjumbotromなどはこの直下に置くこと

section.container
    .container継承
    max-widthのresponsive設定
    余白無し
    .unlimit-widthで、fullhdのときmax-widthなし
    bulmaの margin: 0 auto を打ち消している

.content
    .content継承
    左右下にpaddingがある（padding: 0 2rem 2rem 2rem;）
    :last-child に、padding-bottom:$page-padding-bottom;


コンテンツ内のz-indexは、900未満であること

----------------------------------------------------------
JavaScriptでやっていること
    階層メニュークリック時のexpand（展開／折りたたみ）
    section.container の上マージンを、ヘッダー(nav.header)の高さに合わせる
    ページtopへ移動するボタンの表示と動き
    ハンバーガーボタンのクリックとモーダルメニュー（tablet以下のとき）
    block-ui（おまちください的なやつ）の表示／非表示

    下記のelementはJSで動的に作っている
        ・モーダルメニュー（メニューのnodeをコピって複製）
        ・ページtop移動ボタン
        ・block-ui

CSSでやっていること
    メニュー、メイン、ヘッダのResponsive
    ハンバーガーボタンの表示／非表示
    block-uiのstyle定義
